Initial Array
We start with our initial car array: ["Benz", "Toyota", "Honda", "BMW"]
car = ["Benz", "Toyota", "Honda", "BMW"]
push() Operation
The push() method adds one or more elements to the end of an array and returns the new length.
car.push("Volvo") adds "Volvo" to the end of the array
unshift() Operation
The unshift() method adds one or more elements to the beginning of an array and returns the new length.
car.unshift("Volks Wagen") adds "Volks Wagen" to the beginning of the array
pop() Operation
The pop() method removes the last element from an array and returns that element.
car.pop() removes the last element from the array
shift() Operation
The shift() method removes the first element from an array and returns that element.
car.shift() removes the first element from the array
Element Modification
You can modify array elements by accessing them with their index.
car[1] = "IVM" changes the element at index 1 to "IVM"
Array Length
The length property returns the number of elements in an array.
car.length returns the current length of the array
